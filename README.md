Parcmetres

Installation
==========

Ouvrir git bash
Se déplacer dans le dossier de votre choix : ex : cd /c/wamp, cd ../Documents (utilisez ls pour savoir le contenu du dossier)
Entrer la ligne de code : git clone git@github.com:nicolasroger17/Parcmetres.git
